# Dataset Card

**Назва проєкту:** NER: імена/організації/локації/дати/гроші (готовий
корпус)

**Задача:** B. Extraction / NER  
**Input:** текст українською  
**Output:** сутності: PER, ORG, LOC, DATE, MONEY, ...

**Джерело даних:** https://github.com/lang-uk/ner-uk

**Обсяг:** 11161 текстів, 14735 сутностей

**Мова/домени:** українська, новини/тексти BRUK та Наші гроші

**Очищення:** нормалізація пробілів, апострофів, заміна URL/EMAIL/PHONE

**Ризики:** дисбаланс типів сутностей, короткі тексти, дублікати, можливі упередження

**План наступного кроку:** видалити дублікати, дуже короткі та сміттєві рядки
